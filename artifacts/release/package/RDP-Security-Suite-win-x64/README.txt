RDP Security Suite (win-x64)

Includes:
- WinService.exe (self-contained service binary)
- RDPSecurityViewer.exe + runtime files
- install_service.bat / uninstall_service.bat

Install service:
1) Run install_service.bat as Administrator.
2) Check: sc query RDPSecurityService

Run viewer:
- Start RDPSecurityViewer.exe
