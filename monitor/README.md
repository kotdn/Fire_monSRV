# RDP Security Monitor

GUI-утилита для мониторинга RDPSecurityService в реальном времени.

## 🎯 Функции

### Панели мониторинга:

1. **SERVICE STATUS** (Статус службы)
   - Текущее состояние службы (Running/Stopped)
   - Тип запуска (Automatic/Manual)
   - Кнопки управления: Start/Stop/Refresh

2. **CONFIGURATION** (Конфигурация)
   - RDP порт
   - Уровни блокировок (количество попыток → время блокировки)
   - Читается из `C:\ProgramData\RDPSecurityService\config.json`

3. **BLOCKED IPs** (Заблокированные IP)
   - Список всех IP, заблокированных firewall-правилом
   - Кнопка "Clear All Blocks" — очистить все блокировки
   - Обновляется автоматически каждые 5 секунд

4. **LIVE LOGS** (Логи в реальном времени)
   - Отображение событий из:
     - `service.log` — события службы
     - `access.log` — попытки доступа
     - `block_list.log` — блокировки IP
   - FileSystemWatcher отслеживает изменения файлов
   - Автоскролл к последним событиям

## 🚀 Запуск

### Из исходников:
```powershell
cd monitor
dotnet run
```

### Готовый exe:
```powershell
.\bin\Debug\net8.0-windows\RDPMonitor.exe
```

## 📦 Требования

- .NET 8.0 Runtime (Desktop)
- Windows OS
- Служба RDPSecurityService должна быть установлена

## 🔧 Основные операции

### Запуск/остановка службы
Кнопки "Start Service" / "Stop Service" выполняют команды с проверкой прав администратора.

### Очистка блокировок
Кнопка "Clear All Blocks":
- Очищает файл `block_list.log`
- Удаляет firewall-правило `RDP_BLOCK_ALL`
- Требует подтверждения

### Просмотр папки логов
Кнопка "Open Logs Folder" открывает `C:\ProgramData\RDPSecurityService\` в Explorer.

## 📝 Логи

Монитор отображает 3 типа событий:

- **[SERVICE]** — события самой службы (старт/стоп, ошибки)
- **[ACCESS]** — попытки входа RDP (успешные и неудачные)
- **[BLOCK]** — заблокированные IP с временем и причиной

## ⚙️ Технологии

- **WinForms** — GUI
- **PowerShell SDK** — выполнение команд управления firewall
- **ServiceController** — управление Windows-службами
- **FileSystemWatcher** — мониторинг изменений логов
