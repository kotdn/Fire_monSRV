## 🔗 Маршруты

### Вход/Выход
```
GET  /account/login              → Форма логина
POST /account/login              → Обработка логина
POST /account/logout             → Выход
```

### Главная страница
```
GET /home                         → Показать все сообщения (авторизованный пользователь)
```

### Администрирование
```
GET  /admin                       → Админпанель (только админ)
POST /admin/addmessage           → Добавить сообщение
POST /admin/updatemessage/{id}   → Обновить сообщение
POST /admin/deletemessage/{id}   → Удалить сообщение
```

## 📊 Сравнение SQL запросов

### PHP версия
```php
$pdo = getPDO();
$stmt = $pdo->prepare('SELECT * FROM USERS WHERE username = ?');
$stmt->execute([$username]);
$user = $stmt->fetch(PDO::FETCH_ASSOC);
```

### .NET версия
```csharp
var user = await _context.Users
    .FirstOrDefaultAsync(u => u.Username == username);
```

Явно лучше: типизирован, безопаснее от SQL-инъекций, более читаемо.

## 🧪 Тестирование

Ручное тестирование:

1. **Логин с правильными данными:**
   - Вход как `admin` — ✅ должен перейти на `/home`
   - Вход как `user` — ✅ должен перейти на `/home`

2. **Логин с неправильными данными:**
   - Ввести неверный пароль — ✅ должна появиться ошибка

3. **Доступ к админке:**
   - Как администратор — ✅ должна открыться админпанель
   - Как пользователь — ✅ должна быть ошибка Forbid

4. **Управление сообщениями:**
   - Добавить сообщение — ✅ должно появиться в списке
   - Обновить сообщение — ✅ текст должен измениться
   - Удалить сообщение — ✅ сообщение должно исчезнуть
